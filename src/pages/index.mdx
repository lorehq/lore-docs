## Start In Minutes

1. Install Lore
2. Launch `lore`
3. Choose `Create new project directory` or `Initialize current directory`
4. If needed, enter a project name
5. Select at least one supported tool
6. Confirm the path and selection
7. Finish the flow and enter the TUI

![Lore welcome flow starts with project creation or initialization.](/lore-docs/screenshots/welcome-choice.png)

The welcome flow walks you through naming, tool selection, confirmation, and the final handoff into the full TUI.

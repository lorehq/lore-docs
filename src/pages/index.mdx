The AI coding agent harness — a persistent knowledge layer for AI coding agents.

Source:     https://github.com/andrewloux/lore

## Skills

- `/lore-capture` — review session work, create fieldnotes, sync platform copies
- `/lore-status` — show instance health: version, hooks, skills, agents, active work
- `/lore-create-fieldnote` — capture a non-obvious snag or environmental gotcha
- `/lore-create-skill` — capture a reusable procedure as a skill
- `/lore-consolidate` — deep repo-wide health check

## Primitives

- **Rules** — behavioral constraints loaded into agent context
- **Skills** — reusable procedures for recurring tasks
- **Fieldnotes** — snags, gotchas, and env quirks captured for reuse
- **Runbooks** — multi-step operational procedures
- **Agents** — static delegation patterns for recurring workflows

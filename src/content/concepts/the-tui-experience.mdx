# The TUI Experience

Lore is designed as a rich terminal application with mouse support.

The TUI is the main product experience.

## What That Looks Like

- clickable tabs
- clickable panes and controls
- visible project state and generated output
- a guided welcome flow when no Lore project exists yet

You can still use the keyboard, but Lore is built to be readable and usable as a full interface.

![Lore Projections view with tabs, panes, and visible composition state.](/lore-docs/screenshots/projections-overview.png)

This is the main working view after setup. It shows project state, enabled bundles, and generated output in one place.

## Why It Matters

The TUI makes setup and day-to-day use easier because you can see what is active, what changed, and what Lore will generate.

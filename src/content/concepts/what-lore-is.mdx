# What Lore Is

Lore lets you keep one AI coding setup across supported tools.

Instead of setting up each tool separately, you set things up once and let Lore apply them where needed.

## What You Manage In Lore

- skills
- rules
- agents
- MCPs
- hooks

Lore also gives you:

- global and project scoping
- reusable bundles
- a TUI for setup, inspection, and discovery

## What Lore Is Not

- not a replacement for the coding tools you already use
- not a hosted AI IDE or chat app

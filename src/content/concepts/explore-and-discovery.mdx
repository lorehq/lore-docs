# Explore and Discovery

Use `Explore` to browse bundles, skills, and MCP servers you can add to your setup.

## Tabs In Explore

- `Bundles` for browsing installed and available bundles
- `SkillsMP` for browsing skills from the SkillsMP ecosystem
- `skills.sh` for browsing skills from the Agent Skills ecosystem
- `Smithery` for browsing MCP servers

## Bundles

The `Bundles` tab shows installed bundles and bundles you can add.

It includes actions like update, remove, and view details.

![Explore tab showing the Bundles sub-tab with bundle cards and actions.](/lore-docs/screenshots/explore-bundles.png)

## SkillsMP and skills.sh

The `SkillsMP` and `skills.sh` tabs both help you find skills, but they come from different ecosystems.

Use them when you want to find a skill and add it to the right scope.

![SkillsMP sub-tab with searchable skill cards and add actions.](/lore-docs/screenshots/explore-skillsmp.png)

![skills.sh sub-tab with ranked skill results and add actions.](/lore-docs/screenshots/explore-skillssh.png)

## Smithery

The `Smithery` tab is for MCP server discovery.

Use it when you want to browse available MCP servers and add them to Lore.

![Smithery sub-tab with MCP server cards and add actions.](/lore-docs/screenshots/explore-smithery.png)

## When To Use Explore

Use `Explore` when you want to browse and add bundles, skills, or MCP servers without editing files by hand.

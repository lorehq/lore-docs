# Getting Started

Use these pages to install Lore and set up your first project.

Most people start in the TUI. If you prefer commands, you can also use `lore create` or `lore init`.

## Quick Start

1. Install Lore
2. Launch `lore`
3. Choose `Create new project directory` or `Initialize current directory`
4. If creating a project, enter a project name
5. Select at least one supported tool
6. Confirm and finish the flow
